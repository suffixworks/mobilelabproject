Design System — สกัดจากเว็บเดิม mobilelabproject.com เพื่อใช้เป็นฐานทำ WordPress block theme ใหม่ · ทุก token เปลี่ยนที่เดียวใน :root / theme.json

01 — Logo

wordmark "Mobile Lab." ตัว o เป็น ink-splat · ดึงจาก mobile_lab_final_logo.jpg เดิมแล้ว vectorize เป็น SVG พื้นโปร่ง (คมทุกขนาด/retina) ใช้ currentColor จึงพลิกสีตามพื้นได้

ไฟล์ที่ส่งให้: mobile-lab-logo.svg (เวกเตอร์—ใช้บนเว็บ), mobile-lab-logo.png + @2x (พื้นโปร่ง—fallback) · clear space รอบโลโก้ ≥ ความสูงตัว "M" · ขนาดเล็กสุดแนะนำ 24px

04 — Layout · Content width

เนื้อหาทุกหน้า จัดกึ่งกลางใต้ nav ความกว้างคอลัมน์ = 607px (เท่าเว็บเดิม, ตรงกับ #wrapper/#wrapper2) · WordPress ตั้ง contentSize: 607px · รูป/hero ปล่อย full-bleed ได้

◄ 607px — content column (centered) ►

ทุกบล็อกเนื้อหา (About, ผลงาน, ข่าว) วางในคอลัมน์นี้ จัดกึ่งกลางหน้าจอ ส่วนรูปภาพขยายเต็มความกว้างคอลัมน์ (max-width:100%) แล้ว WordPress เสิร์ฟ responsive srcset ให้เอง
